# Валидация переноса образа Snow → Mosaic

**Дата:** 2026-02-05
**Статус:** Draft
**Цель:** Проверить корректность переноса образа с поверхности снега (phels) на мозаику детектора (moshits) через оптическую систему SPHERE-3.

---

## 1. Обзор и цели

**Цель:** Создать набор синтетических тестовых образов (phels-файлов) с известными геометрическими параметрами, прогнать их через Geant4 симуляцию, и количественно оценить качество переноса на мозаику.

### Три типа тестовых образов

| Тип | Параметры | Что проверяем |
|-----|-----------|---------------|
| **Точечный источник** | Одна точка (x₀, y₀) на снегу, N фотонов | Размер PSF (Point Spread Function), смещение центроида |
| **Кольцо** | Окружность радиуса R, центр (x₀, y₀) | Радиальная симметрия, сохранение радиуса |
| **Сетка** | Матрица точек NxM с шагом Δ | Геометрическое отображение, дисторсия |

### Три аспекта валидации

1. **Геометрия:** Сравнение (x, y) входа → (x, y) выхода. Метрики: смещение центроида, масштабный коэффициент, дисторсия.

2. **Время:** Сравнение t₀ (вход) → t (выход). Метрики: временна́я задержка, размытие (σ_t).

3. **Эффективность:** N_вход / N_выход. Сравнение с аналитической оценкой (геометрия зеркала × отражаемость × телесный угол PMT).

### Выходные артефакты

- Генератор синтетических phels-файлов
- Скрипт анализа moshits с метриками
- Отчёт с графиками и таблицами

## 3. Анализ результатов (moshits → метрики)

### Структура анализатора

```python
class TransferValidator:
    def __init__(self, phels_path: Path, moshits_path: Path):
        self.phels = load_phels_data(phels_path)
        self.moshits = load_data(moshits_path)
```

### 3.1 Геометрические метрики

| Метрика | Формула | Интерпретация |
|---------|---------|---------------|
| **Смещение центроида** | Δx = x̄_out - x̄_in, Δy = ȳ_out - ȳ_in | Систематическая ошибка позиционирования |
| **Масштаб** | k = R_out / R_in | Коэффициент увеличения оптики |
| **PSF (σ)** | σ = √(⟨(x-x̄)²⟩ + ⟨(y-ȳ)²⟩) | Размытие точечного источника на мозаике |
| **Эллиптичность** | e = (σ_x - σ_y)/(σ_x + σ_y) | Анизотропия PSF |
| **Дисторсия** | D(r) = (r_measured - r_ideal) / r_ideal | Радиальное искажение (для сетки) |

### 3.2 Временны́е метрики

| Метрика | Формула | Интерпретация |
|---------|---------|---------------|
| **Задержка** | Δt = t̄_out - t̄_in | Время пролёта через оптику |
| **Размытие** | σ_t = std(t_out - t_in) | Хроматическая/геометрическая дисперсия |
| **Временно́й профиль** | Гистограмма (t_out - t_in) | Форма импульсного отклика |

### 3.3 Эффективность

| Метрика | Формула | Интерпретация |
|---------|---------|---------------|
| **Полная эффективность** | ε = N_out / N_in | Доля детектированных фотонов |
| **Эффективность vs R** | ε(R) = N_out(R) / N_in(R) | Зависимость от расстояния до оси |
| **Сравнение с теорией** | ε_теор = Ω_зеркала × ρ_отраж × Ω_PMT | Аналитическая оценка |

## 4. Аналитические предсказания

### Оптическая схема SPHERE-3

Из DetectorConstruction:
- Зеркало: сфера, R ≈ 1751 мм, отражаемость 85-88%
- Мозаика: сфера, R = 862.5–1100 мм, 379 кластеров × 7 пикселей
- Фокусное расстояние: F ≈ 330 мм (DETECTOR_FOCAL_LENGTH)
- Высота детектора: H = 500 или 1000 м

### 4.1 Геометрическое отображение

Для сферического зеркала, точка на снегу (X, Y) отображается на мозаику:

```
x_mosaic = -F × (X / H)
y_mosaic = -F × (Y / H)
```

**Ожидаемый масштаб:** k = F / H
- При H = 500 м: k = 0.33 / 500 = 6.6 × 10⁻⁴
- При H = 1000 м: k = 3.3 × 10⁻⁴

**Инверсия:** знак минус — образ перевёрнут

### 4.2 Теоретическая PSF

Размер пятна определяется:
- Дифракция (пренебрежимо мала для λ ~ 400 нм)
- Аберрации зеркала (сферическая, кома)
- Размер пикселя PMT: 3×3 мм

**Ожидаемая PSF:** σ ≈ 1–3 мм (доминирует размер пикселя)

### 4.3 Теоретическая эффективность

```
ε_теор = ε_геом × ρ_mirror × ε_PMT × (1 - ε_shadow)
```

| Компонент | Значение | Источник |
|-----------|----------|----------|
| ε_геом | Ω_mirror / 2π | Телесный угол зеркала |
| ρ_mirror | 0.85–0.88 | Отражаемость Al |
| ε_PMT | cos(ξ)^1.093 | Угловая чувствительность |
| ε_shadow | ~5% | Затенение конструкцией |

### 4.4 Теоретическая задержка

```
Δt = (H + L_mirror + L_mosaic) / c
```

Где L_mirror, L_mosaic — оптический путь внутри детектора (~2–3 м).

## 8. Критерии успеха и edge cases

### Критерии прохождения валидации (Pass/Fail)

| Категория | Метрика | Критерий Pass | Критерий Warning |
|-----------|---------|---------------|------------------|
| **Геометрия** | Смещение центроида | \|Δ\| < 1 мм | 1–3 мм |
| | Масштаб k | \|k_meas/k_theory - 1\| < 5% | 5–10% |
| | PSF σ | σ < 5 мм | 5–10 мм |
| | Дисторсия D(r) | D < 3% для r < 300м | 3–5% |
| **Время** | Задержка Δt | \|Δt_meas - Δt_theory\| < 50 нс | 50–100 нс |
| | Размытие σ_t | σ_t < 30 нс | 30–50 нс |
| **Эффективность** | ε общая | \|ε_meas/ε_theory - 1\| < 10% | 10–20% |
| | ε(R) форма | χ²/ndf < 2 | 2–5 |

### Edge cases и потенциальные проблемы

| Случай | Проблема | Как обнаружить | Действие |
|--------|----------|----------------|----------|
| Фотоны вне апертуры | Точка слишком далеко от оси | N_out = 0 | Ограничить R < 400м |
| Перекрытие объёмов | Geant4 warning о overlaps | Логи симуляции | Проверить геометрию |
| Crosstalk искажает PSF | σ_measured >> σ_expected | Анализ без crosstalk (origin=1) | Фильтровать по origin |
| Отражения от Hood | Ложные хиты | Аномальные t или направления | Фильтр по vz < 0 |
| Недостаточная статистика | Большие ошибки метрик | σ_metric / metric > 10% | Увеличить N_photons |
| Краевые пиксели | Неполное покрытие | Хиты на границе мозаики | Исключить крайние кластеры |

### Диагностические проверки

```python
def sanity_checks(phels_df, moshits_df) -> List[str]:
    warnings = []

    # 1. Есть ли вообще хиты?
    if len(moshits_df) == 0:
        warnings.append("FAIL: No hits detected")

    # 2. Эффективность в разумных пределах?
    eff = len(moshits_df) / len(phels_df)
    if eff < 0.01:
        warnings.append(f"WARNING: Very low efficiency {eff:.2%}")

    # 3. Хиты внутри мозаики?
    r_max = np.sqrt(moshits_df['x']**2 + moshits_df['y']**2).max()
    if r_max > 150:  # мм
        warnings.append(f"WARNING: Hits outside mosaic r={r_max:.1f}mm")

    # 4. Времена в ожидаемом окне?
    t_range = moshits_df['t'].max() - moshits_df['t'].min()
    if t_range > 1000:  # нс
        warnings.append(f"WARNING: Large time spread {t_range:.0f}ns")

    return warnings
```
